# FitBench
iOS fitness benchmarking app — ranks your runs by age-graded percentile.

## Stack
- **Frontend:** React Native + Expo + TypeScript
- **Backend:** FastAPI + PostgreSQL (Supabase)
- **Integrations:** Apple HealthKit, Strava OAuth2

## Structure
/mobile   → React Native app
/backend  → FastAPI API
